# Security Policy

This is a proprietary desktop application maintained by Helpers Technologies.

## Sensitive Files

Do not commit or share:

- Production license key material
- `.pfx`, `.p12`, `.pem`, `.key`, or signing backup files
- Generated release installers and client USB packages
- Customer databases, backups, license tokens, or support codes

The real `electron/license-public-key.cjs` file is local-only. The repository includes `electron/license-public-key.example.cjs` as a safe template.

## Reporting

Report security issues privately to Helpers Technologies through the official support channel.
