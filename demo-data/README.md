# بيانات الديمو — Demo Data

ملفات CSV جاهزة للاستيراد داخل السيستم لعرض النظام على العملاء ببيانات واقعية.

## طريقة الاستيراد

1. افتح السيستم وسجّل الدخول.
2. من القائمة الجانبية: **استيراد بيانات**.
3. تبويب **المنتجات** ← «اختر ملف CSV» ← اختر `ديمو_منتجات.csv` ← «استيراد».
4. تبويب **العملاء** ← «اختر ملف CSV» ← اختر `ديمو_عملاء.csv` ← «استيراد».

## محتوى الملفات

- **ديمو_منتجات.csv** — 32 منتج جملة (بقالة، زيوت، معلبات، منظفات، مشروبات...) بأكواد P001–P032
  وأسعار شراء/جملة/تجزئة واقعية. آخر 5 أصناف كمياتها منخفضة أو صفر عمداً
  (P028–P032) لإظهار تنبيهات «منتجات قليلة المخزون» و«نفد المخزون» أثناء العرض.
- **ديمو_عملاء.csv** — 15 عميل (سوبر ماركت وبقالات) بأرقام موبايل وعناوين وملاحظات.

## ملاحظات

- الملفات بترميز UTF-8 مع BOM — تفتح في Excel بالعربي مباشرة وتُقبل في صفحة الاستيراد كما هي.
- الموردون والسائقون وفواتير الديمو غير قابلة للاستيراد من CSV — تُدخل يدوياً من صفحاتها
  (الاستيراد يدعم المنتجات والعملاء فقط).
- المنتجات المكررة الكود يتم تخطيها تلقائياً عند إعادة الاستيراد.
