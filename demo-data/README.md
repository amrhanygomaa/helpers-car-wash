# بيانات الديمو — Demo Data

ملفات جاهزة للاستيراد داخل السيستم لعرض النظام على العملاء ببيانات واقعية.

---

## 📦 ديمو تاجر التسالي والحلاويات (الملف الشامل)

### `ديمو_تسالي_وحلاويات.json`

ملف JSON كامل يحتوي على **سنة كاملة من البيانات** (يوليو 2025 – يونيو 2026)
لتاجر توريد تسالي وحلاويات مصري — **"البركة لتجارة التسالي والحلويات"**.

#### محتويات الملف:

| البيان                | العدد   | الوصف                                                    |
|-----------------------|---------|----------------------------------------------------------|
| المنتجات              | 50      | تسالي، شيكولاتة، شيبسي، بسكويت، حلاويات، مشروبات، موسمي |
| الموردون              | 8       | مصانع وشركات تسالي وشيكولاتة وسناكس (بعمولات ربع سنوية)  |
| العملاء               | 20      | سوبر ماركت، بقالات، أكشاك — القاهرة والجيزة والقليوبية   |
| السائقون              | 4       | سائقين للتوصيل                                           |
| فواتير الشراء         | ~257    | فواتير شراء أسبوعية من كل مورد                            |
| فواتير البيع          | ~1,170  | 3-7 فواتير يومياً (أكثر في رمضان والصيف)                 |
| مرتجعات البيع         | ~10     | بضاعة تالفة أو قريبة انتهاء الصلاحية                     |
| مرتجعات الشراء        | ~4      | خطأ توريد أو بضاعة تالفة                                  |
| حركات المخزون         | ~6,800+ | شراء + بيع + تعديلات + مرتجعات                            |
| حركات الصندوق         | ~1,240  | تحصيلات + مدفوعات + إيجار + مصاريف + إيداعات              |
| عروض أسعار            | 6       | 3 محولة لفواتير + 3 مسودات                               |
| جرد                   | 3       | جرد ربع سنوي (مُطبَّق)                                  |
| سجل المراجعة          | 8       | عينة أحداث (إنشاء، إلغاء، تعديل)                        |
| فواتير ملغاة          | ~4      | فواتير بيع ملغاة                                          |
| إجمالي المشتريات      | ~8.1M   | ج.م                                                      |
| إجمالي المبيعات       | ~17.3M  | ج.م                                                      |

#### الفئات الممثلة:
- 🥜 تسالي ومكسرات (لب، فول سوداني، كاجو، لوز)
- 🍫 شيكولاتة (سادة، حليب، ويفر، كرات)
- 🍿 شيبسي وسناكس (شيبسي، بفك، كيرلي)
- 🍪 بسكويت وكيك (محشو، سادة، كرواسون)
- 🍬 حلاويات (حلاوة طحينية، حلقوم، ملبس، بسبوسة، كنافة)
- 🍭 لبان وحلوى (لبان، بونبون، تافي، مصاصات)
- 🧃 مشروبات (عصائر، مياه، غازيات)
- 🧇 وافل وكريب
- 🌙 موسمي / رمضان (ياميش، قمر الدين، تمر)

#### Use Cases مغطاة:
- ✅ فواتير شراء (مدفوعة بالكامل / جزئياً / غير مدفوعة)
- ✅ فواتير بيع (كاش / آجل — جملة / تجزئة)
- ✅ خصومات على فواتير البيع
- ✅ مرتجعات بيع (استرداد نقدي / خصم من الرصيد)
- ✅ مرتجعات شراء
- ✅ حركات مخزون (شراء / بيع / تعديل يدوي)
- ✅ صندوق النقدية (تحصيلات / مدفوعات / مصاريف / إيداعات)
- ✅ إيجار شهري + مصاريف تشغيلية
- ✅ عروض أسعار (مسودة + محولة لفواتير)
- ✅ جرد ربع سنوي
- ✅ منتجات قليلة المخزون / نفدت (SN048, SN049, SN050)
- ✅ منتجات بتاريخ صلاحية
- ✅ عمولات موردين (ربع سنوية بشرائح)
- ✅ سائقين توصيل مربوطين بالفواتير
- ✅ اتجاه شحن (قبلي / بحري)
- ✅ طرق دفع متعددة (كاش / بنكي / فودافون / إنستاباي)
- ✅ فواتير ملغاة
- ✅ سجل مراجعة

#### طريقة الاستيراد:
1. افتح السيستم وسجّل الدخول.
2. من القائمة الجانبية: **الإعدادات**.
3. في قسم النسخ الاحتياطي: **«استيراد نسخة احتياطية»**.
4. اختر ملف `ديمو_تسالي_وحلاويات.json`.
5. سيتم تحميل جميع البيانات تلقائياً.

> ⚠️ **تنبيه**: الاستيراد يستبدل جميع البيانات الحالية في السيستم. تأكد من أخذ نسخة احتياطية أولاً.

## 📋 ملفات CSV (استيراد منتجات وعملاء فقط)

### ديمو التسالي (CSV)
- **ديمو_تسالي_منتجات.csv** — 50 منتج تسالي وحلاويات بأكواد SN001–SN050
- **ديمو_تسالي_عملاء.csv** — 20 عميل (سوبر ماركت وبقالات وأكشاك)

### ديمو البقالة العام (CSV)
- **ديمو_منتجات.csv** — 32 منتج جملة (بقالة، زيوت، معلبات، منظفات...) بأكواد P001–P032
- **ديمو_عملاء.csv** — 15 عميل

#### طريقة استيراد CSV:
1. افتح السيستم وسجّل الدخول.
2. من القائمة الجانبية: **استيراد بيانات**.
3. تبويب **المنتجات** ← «اختر ملف CSV» ← اختر الملف ← «استيراد».
4. تبويب **العملاء** ← «اختر ملف CSV» ← اختر الملف ← «استيراد».

## ملاحظات

- الملفات بترميز UTF-8 — تفتح في Excel بالعربي مباشرة.
- ملف JSON الشامل يحتوي على **جميع** أنواع البيانات (منتجات، عملاء، موردون، فواتير، مرتجعات، صندوق، جرد، عروض أسعار).
- ملفات CSV تدعم استيراد **المنتجات والعملاء فقط** — باقي البيانات تحتاج ملف JSON.
- المنتجات المكررة الكود يتم تخطيها تلقائياً عند إعادة الاستيراد.
- الموردون والسائقون لا يدعمون الاستيراد من CSV — استخدم ملف JSON.
