# نظام المخزون والمبيعات — Warehouse & Sales Demo

نظام إدارة متكامل لمستودع/توزيع صغير: المنتجات، المخزون، فواتير المشتريات والمبيعات، الخزينة، التنبيهات، والتقارير.
مبني بـ React + TypeScript + Vite + Tailwind + shadcn-style components + recharts، مع حفظ كامل في localStorage.

## تشغيل المشروع

```bash
npm install
npm run dev
```

المتصفح: http://localhost:5173 — الدخول: `admin` / `demo` (ديمو فقط).

## الأوامر

```bash
npm run dev       # خادم التطوير
npm run build     # بناء الإنتاج
npm run lint      # فحص الأكواد
npm run preview   # معاينة البناء
```

## المميزات

- **لوحة تحكم** بإحصائيات ومخططات (مبيعات/مشتريات/مخزون).
- **المنتجات**: CRUD كامل، بحث/فلترة/ترتيب، سجل حركات، تاريخ صلاحية.
- **المخزون**: جدول مخزون، تنبيهات نقص/قرب انتهاء/منتهي، ضبط يدوي مع سبب.
- **الموردين والعملاء**: CRUD، أرصدة، سجل فواتير.
- **فواتير المشتريات**: إنشاء، طباعة A4، تسجيل دفعات — تزيد المخزون تلقائياً.
- **فواتير المبيعات**: إنشاء (نقدي/آجل)، سائق اختياري، طباعة A4، تسجيل تحصيل، إلغاء — تخصم المخزون، مع حظر الحفظ عند نقص الكمية.
- **التنبيهات**: نقص المخزون، قرب الانتهاء، المنتهي، عملاء/موردين لديهم أرصدة.
- **الخزينة**: رصيد افتتاحي، تلقائي من المبيعات/المشتريات، قيود يدوية.
- **التقارير**: مبيعات، مشتريات، مخزون، أرصدة — مع مخططات وفلاتر.
- **الإعدادات**: اسم الشركة، ذيل الفاتورة، العملة، حد أدنى، مقاس الطباعة، إعادة التعيين.

## البنية

```
src/
  components/ui      مكونات (Button, Card, Table, Dialog, Drawer, Toast, ...)
  components/layout  Sidebar, Topbar, AppLayout
  features/          نماذج (منتجات، فواتير)
  pages/             18 صفحة كاملة
  lib/               utilities + localStorage
  store/             AppContext (الحالة + المنطق)
  data/              البيانات التجريبية (عربية)
  types/             الأنواع المشتركة
```

## افتراضات

- التطبيق ديمو واجهي بالكامل: لا يوجد API. الحفظ في localStorage.
- العملة: ج.م (قابلة للتغيير من الإعدادات).
- تسجيل الدخول وهمي (`admin` / أي كلمة مرور).
